**📊 E-Commerce Sales Analytics Dashboard (Excel)**

**Overview**

This project is an Excel-based E-Commerce Sales Analytics Dashboard designed to transform raw sales data into meaningful business insights. The dashboard helps analyze customer behavior, sales performance, order trends, discount impact, and delivery efficiency through interactive visualizations and KPIs.

**Objectives**
1. Analyze customer purchasing patterns.
2. Track sales and revenue performance.
3. Identify top-performing customers.
4. Measure the impact of discounts and coupon codes.
5. Monitor delivery timelines and late deliveries.
6. Create an interactive dashboard for data-driven decision-making.
7. Dataset Information

**The dataset contains 500+ e-commerce transactions and includes:**

1. Customer details
2. Order information
3. Product data
4. Pricing and quantity
5. Coupon and discount details
6. Delivery information
7. Revenue and final sales amounts
   
**Key KPIs**
1. Total Unique Customers
2. Total Orders
3. Total Revenue
4. Average Order Value
5. Top Customers by Sales
6. Discount Analysis
7.Late Delivery Percentage
8. Average Delivery Time
9. Tools & Techniques Used
10. Microsoft Excel

**Project Outcome**
The dashboard provides a comprehensive view of e-commerce performance, enabling users to identify trends, monitor KPIs, and make informed business decisions using interactive Excel-based reporting.
